FANUC Stitch Weld

This is a Stitch Welding plug-in provided by FANUC.


The User Defines:

  • Start and End point.
  • Weld Length.
  • Weld Pitch.
  • Weld Procedure.
  • Weld Schedule.
  • Weld Speed and Speed Unit.
  • TPP Name for Program.

How it Works:

  1. Drag the instruction into the timeline.
  2. Enter parameters on the Details Screen.
  3. When the instruction is called, a TP program will be generated.
  4. The user can call the program or run it at a different time.
