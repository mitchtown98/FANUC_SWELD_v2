FANUC Stitch Weld

This is a sample plugin provided by FANUC.

Description

  • This is a sample plugin to automate creating a Stitch Weld.
  • Not for release.

Files Loaded: with prefix "IPL_FANUC"

  1. SWELD_CFG.PC = Variable Definition
  2. SWELD_SET.PC = Passes Arguments to create Stitch Segments
  3. SWELD_GENERATETP.PC = Generates a TPP from created Segments
  4. SWELD_SERVER.PC = Handles utility methods for the instruction
  5. SWEKD_KLSET.TP = Subprogram to pass arguments to SWELD_SET.PC

Required:

!!! LR ArcTool H574 is required for this plugin to install !!!
